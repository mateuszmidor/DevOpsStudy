# customizing with command line args and python code

## Run
```sh
tilt up -- Jenny White
```

```text
Running cmd: echo Welcome, Jenny White
Welcome, Jenny White
```

You can change the cmd line args when tilt is already up:
```sh
tilt args -- Johny Bravo
```

```text
Running cmd: echo Welcome, Johny Bravo
Welcome, Johny Bravo
```
